# Пользовательские Сценарии

## Покупка

1. Пользователь открывает Telegram-бота.
2. Выбирает тариф и срок.
3. Выбирает платежный метод.
4. После успешной оплаты бот создает или продлевает клиента в 3X-UI.
5. Пользователь переходит в подключение и добавляет подписку в Happ.

## Подключение

Активный onboarding рассчитан на Happ:

- iOS;
- Android;
- Windows.

На каждой платформе пользователь видит один и тот же понятный набор действий:

- подключить основную подписку;
- добавить режим "РФ-сервисы напрямую, остальное через VPN";
- подключить подписку обхода БС, если она включена в тариф;
- скачать Happ.

## Подписка Обхода БС

Подписка обхода БС доступна только на тарифах с `includes_additional_profile: true`.

Порядок для пользователя:

1. Сначала нажать **Подписка обхода БС - рекомендуется**.
2. Если не подошло, нажать **Подписка обхода БС - запасной вариант**.
3. Если кнопка открыла описание тарифа, значит текущий тариф не включает обход БС.

## Апгрейд

Если пользователь на тарифе без обхода БС, бот может предложить апгрейд на связанный тариф через `upgrade_from`.

Важно: апгрейд должен быть понятен как изменение тарифного доступа, а не как "техническая настройка".
