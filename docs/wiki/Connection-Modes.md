# Режимы Подключения

## Основная Подписка

Основная подписка - главный VPN-профиль пользователя.

- Endpoint: `/sub/{vpn_id}`.
- Доступна активным пользователям.
- Добавляется в Happ через кнопку **Подключить основную подписку**.

## РФ-Сервисы Напрямую, Остальное Через VPN

Это отдельная Happ routing-настройка.

Поведение:

- российские домены и российские IP идут напрямую;
- зарубежные сервисы идут через VPN;
- основная подписка не смешивается с routing-настройкой.

Кнопка в боте: **РФ-сервисы напрямую, остальное через VPN**.

## Подписка Обхода БС

БС - "белые списки". Это отдельная подписка для тарифов, где обход БС включен.

| Вариант | Endpoint | Когда использовать |
| --- | --- | --- |
| Рекомендуемый | `/wl-filtered/{vpn_id}` | сначала, основной пользовательский вариант |
| Запасной | `/wl/{vpn_id}` | если рекомендуемый вариант не подошел |

## Что Не Показываем Пользователю

- выбор альтернативных Android-клиентов;
- raw-конфигурацию отдельной кнопкой;
- технические названия источников в пользовательских кнопках.

Технический raw-режим может оставаться в backend для совместимости и диагностики, но не должен быть первым пользовательским сценарием.
